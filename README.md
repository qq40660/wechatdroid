WechatDroid
===========

This project is no longer maintained due to the newly announced official API

可以自主学习的微信机器人

Dependecies
------
python 2.7, requests, sqlite3

Usage
------
* 安装requests, `pip install requests`
* run it and have fun `python wechatdroid.py`
