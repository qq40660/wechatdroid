WechatDroid
===========

可以自主学习的微信机器人

Dependecies
------
python 2.7, requests, sqlite3

Usage
------
* 安装requests, `pip install requests`
* 在wechatdroid.py所在目录建立sqlite数据库wechatdroid.db
* 使用wechatdroid.sql初始化
* run it and have fun `python wechatdroid.py`
